# How to contribute to poster sets

## Steps

- Create the art
- Fork the repo 🍳
- Create a new folder called with the name of the set in `posters/`
- Place each poster in the created folder with the following names:
  - `poster.png` for the Show poster
  - `SeasonXX.png` where XX is the Seacson number (two digits, Specials => `Season00.png`)
- Create a pull request 🍳
- Profit

## 🍳 Cannot use github or dev tools

if you don't know how to do any of the steps listed with 🍳, it doesn't matter.

Just prepare what you can and find a way to contact me either via github issues or through the One Pace's discord.

You've done the hard part, I'll help you with the rest :)
